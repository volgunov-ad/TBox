# Changelog

## 0.9.10 (текущая) — изменения с 0.9.5

### Новые возможности
- Плавающие панели: поддержка 1–3 панелей, выбор активной панели и отдельные настройки
  (набор плиток, размеры, позиция, фон, действие по клику).
- Вкладка CAN: список CAN ID с последними кадрами и экспорт текущих данных в файл.
- Новые данные в разделе «Данные авто»: расширенные параметры CAN, давления и температуры
  колес, температуры в салоне/снаружи, режимы КПП, блокировка окон.
- Циклические данные (ускорения, давление шин, обороты) при включении в экспертных настройках.
- Моточасы двигателя: сохраняются между запусками; двойной тап по плитке сбрасывает счетчик.
- Новая плитка «давление + температура в шинах».

### Улучшения и интерфейс
- Сворачиваемое левое меню с иконками вкладок и запоминанием состояния видимости.
- Автоматический выход из режима редактирования плавающей панели и сохранение размеров/позиции.
- Обновлены иконки меню и вкладок.
- Расширена русская документация (Readme).
