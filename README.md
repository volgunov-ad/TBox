== TBox Monitor для Jetour Dashing

Приложение предназначено для обмена данными между головным устройством и блоком TBox в автомобиле Jetour Dashing. Проект объединяет интерфейс пользователя, фоновую службу и плавающие панели, чтобы мониторить состояние автомобиля и управлять связью с блоком TBox в реальном времени.
== Основные возможности
* Мониторинг связи с TBox: состояние регистрации в сети, уровень сигнала, статус APN и обновление сетевых параметров.
* Отображение данных на плитках (виджетах) в основном интерфейсе и во всплывающих панелях.
* Плавающие панели (1–3): независимые наборы настроек, собственные размеры и положение, включение фона, действие по нажатию.
* Команды управления: перезапуск модема, перезагрузка TBox, команды SUSPEND/STOP, запрет автоматической перезагрузки.
* Логи работы и диагностическая информация, включая отправку AT-команд.
* Экспорт данных и журналов в файл.

== Плавающие панели
В настройках можно выбрать панель (1, 2 или 3), затем настроить набор плиток, размеры и координаты, включить фон и включить/выключить действие при одиночном нажатии. В режиме редактирования панель можно перемещать за края и менять размер за правый нижний угол.
== Разрешения
Для корректной работы приложению нужны:
* Разрешение на отображение поверх других окон — для плавающих панелей.
* Доступ к хранилищу — для экспорта данных в файл.
* Разрешения геолокации — для режима подмены местоположения.
== Хранение настроек
Все настройки сохраняются в DataStore. Плавающие панели имеют собственные конфигурации, что позволяет отдельно настраивать каждую панель.
