# Changelog

## 1.0.0

Изменения с коммита `fa4e261` («Новые виджеты уровня сигнала сети») по текущую версию.

1. **Главный экран и плитки:** встроенные панели дашборда с относительным позиционированием и изменением размера; отдельная вкладка настроек главного экрана; обои из галереи отдельно для светлой и тёмной темы (Crop / Fit); опция открывать главный экран после загрузки ГУ; настраиваемые позиции кнопок «+» и настроек.
2. **Плавающие панели:** выбор панели в настройках, переименование, создание и удаление (в том числе всех); индикатор обрыва связи с TBox по панели; задержка удаления после скрытия; правки сохранения имени и выбора панели.
3. **Виджеты:** скругление углов (shape) и визуальные дефолты для основного дашборда и плавающего; виджет ярлыка приложения и виджет стороннего приложения (App Widget) в плитках.
4. **Музыкальный виджет:** поддержка дополнительных плееров (в т.ч. Bluetooth-телефон, WT Local, Yandex Music auto и др.), открытие штатного плеера по двойному тапу для Bluetooth; опция автозапуска воспроизведения при появлении виджета и опция **«Запускать только при включении двигателя»** (ожидание оборотов двигателя до 2 минут с периодической проверкой RPM).
5. **Учёт поездок:** сохранение поездок в DataStore (до 31 записи), избранное; настройки объёма бака и времени разделения поездок; логика продолжения поездки после короткой остановки и при перезапуске службы; вкладка «Поездки» и виджеты «Текущая поездка» (полный и упрощённый); расход и заправки по уровню топлива с учётом скачков и простоя; кнопка «Завершить» / двойной тап по полному виджету для завершения активной поездки.
6. **Транспорт и сервис:** обмен с TBox через **tbox-proxy** (UDP), обновление зависимости; сопутствующие правки запуска службы и моточасов.
7. **Прочее:** уровень топлива в литрах с настраиваемым объёмом бака; правки строк (длительности поездок, «текущая поездка» и др.); исправление расчёта пробега поездки по одометру (км без лишнего деления на 1000); мелкие исправления UI и тестов.
